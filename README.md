# weather-dashboard

## Purpose
Weather Dashboard is an app to search for current and future weather conditions. When searching for a city it will present the user with the current day's forecast as well as the 5 day forecast. 

## Built With
- HTML
- CSS/Bootstrap
- Javascript
- moment.js
